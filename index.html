What Evid captures

Evid uses the macOS Accessibility API to read visible text on your screen. Here's exactly what it does — and doesn't — do.

What it reads

  • Visible text in your editor, browser, and documents
  • Which apps and windows are open
  • URLs and file paths you're working with
  • AX tree structure — the text layer, not pixels

What it never does

  • No screenshots or screen recording
  • No keylogging
  • No reading password fields
  • No uploading anything to any server

Built-in safeguards

  • Skips private / incognito browser windows
  • Excludes password managers by default
  • Skips banking and login flows
  • You can add custom app exclusions

Session-scoped

  • Only captures while a session is active
  • Nothing is recorded outside of sessions
  • Delete any session at any time
  • All data stored locally on your Mac
